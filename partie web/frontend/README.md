# 📊 Dashboard ServerRoom Guardian

Dashboard React compatible avec le backend loeni-auth-api.

## Fonctionnalités

✅ Authentification (login/logout)
✅ Gestion utilisateurs (validation techniciens)
✅ Gestion salles serveurs
✅ Gestion checklists
✅ Vérifications
✅ Historique

## Installation

Voir INSTALLATION.md

## Configuration

Le fichier .env contient:
```
REACT_APP_API_URL=http://localhost:4000
```

## Pages disponibles

- /login → Connexion
- /dashboard → Vue d'ensemble
- /users → Gestion utilisateurs
- /rooms → Gestion salles
- /checklists → Gestion checklists
- /verifications → Vérifications
- /history → Historique
